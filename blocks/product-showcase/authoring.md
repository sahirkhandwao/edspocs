## Row Types

- **icon**: Use this for the 4-column grid. Provide an image in the second column and the label text in the third.
- **separator**: Use this for the divider row. Provide the text in the second column.
- **carousel**: Use this for slides. Provide the promotional image in the second column. Text content is not supported for these slides.
