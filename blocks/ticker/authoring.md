# Ticker Block

The Ticker block provides a high-level header bar with branding, market data, search, and quick access buttons.

## Authoring Instructions

Create a table with 4 rows. Each row corresponds to a specific part of the ticker.

| Ticker |
| --- |
| ![Logo](https://www.hdfcsec.com/Content/images/logo.png) |
| <ul><li>NIFTY : 25,741.00 58.25(0.23%)</li><li>SENSEX : 83,526.16 249.01(0.30%)</li></ul> |
| Quotes, Nav |
| <ul><li>[OPEN TRADING A/C](https://...)</li><li>[LOGIN](https://...)</li><li>[INVESTRIGHT LOGIN (NEW)](https://...)</li></ul> |

### Table Breakdown

1.  **Row 1 (Logo)**: Insert the logo image.
2.  **Row 2 (Market Data)**: A list of market items. Use the format `NAME : VALUE CHANGE(PERCENT)`.
3.  **Row 3 (Search Placeholder)**: The text to show in the search bar.
4.  **Row 4 (Buttons)**: A list of links.
    -   The first link will be styled as the **Primary (Red)** button.
    -   Subsequent links will be styled as **Secondary (Blue)** buttons.
    -   Add `(NEW)` to the link text to display a "NEW" badge above the button.
